# Assets Directory

This directory contains static assets for the website.

## Images

Add your profile image here with the name `profile.jpg` or `profile.png`.

Recommended:
- Profile photo: 400x400px or larger, square aspect ratio
- File formats: JPG, PNG, or WebP
- File size: Keep under 500KB for optimal loading

## Icons

Custom icons or favicons can be placed here as well.

To add a favicon:
1. Generate a favicon from your image
2. Place `favicon.ico` in the root directory
3. Add this to the `<head>` section of index.html:
   ```html
   <link rel="icon" type="image/x-icon" href="favicon.ico">
   ```
