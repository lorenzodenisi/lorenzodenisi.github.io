## ğŸ¨ Customization

### Adding Your Profile Image

1. Add your photo to the `docs/assets/` directory as `profile.jpg` or `profile.png`
2. Update the HTML in `docs/index.html` if needed

### Updating Content

- **Personal Info**: Edit the hero section in `docs/index.html`
- **Experience**: Update the timeline section with your work history
- **Education**: Modify the education cards
- **Skills**: Add or remove skills in the skills section
- **Contact**: Update email and social media links

### Changing Colors

Edit the CSS variables in `docs/styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #3b82f6;
    /* ... more colors */
}
```

## ğŸŒ Deployment

### GitHub Pages

1. Push your changes to GitHub:
```bash
git add .
git commit -m "Update personal website"
git push origin main
```

2. Enable GitHub Pages:
   - Go to your repository settings
   - Navigate to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/docs" folder
   - Click "Save"

3. Your site will be live at `https://lorenzodenisi.github.io`
